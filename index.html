Vim Shortcuts

Master Vim with these essential shortcuts.

Category Command Description
Movement h, j, k, l Move left, down, up, and right.
w, e Move to the next word's start (w) or end (e).
b Move to the previous word's start.
Visual Mode (Selection) v Start character-based selection.
Copy & Paste y Copy selected area or line.
p, P Paste to the right (p) or left (P) of the cursor.
Search & Replace /word Search for "word" forward.
:%s/old/new/g Replace all occurrences of "old" with "new" in the file.
Save & Exit :w Save the file.
:q Exit without saving (:q! to force quit).
